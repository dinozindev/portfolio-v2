# Portfólio v2

Segunda versão do meu portfólio pessoal, inspirado no Portfolio v4 da [Brittany Chiang](https://github.com/bchiang7/v4).

![App Screenshot](https://imgur.com/qhh7sZC.png)

## Tecnologias usadas
- TypeScript
- React.js
- Tailwind CSS
- Vite

## Instalação

Para rodar a aplicação, instale as dependências:
```
npm install
```

E execute a aplicação com:
```
npm run dev
```
